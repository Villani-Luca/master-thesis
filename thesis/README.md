## How to use this template
- Examine the folder structure to understand how the project is organized.
- Read the comments at the top of the bibliography.bib file.
- Review the examples provided throughout the template; each chapter includes useful guidance on effective thesis writing.
- Start writing and add citations as you go. Do not postpone the bibliography until the end.
- This template does not prescribe how to structure the content of your thesis. Consult your supervisor for guidance.

## Writing standards
- Use consistent terminology and notation, and do not change them throughout the thesis.
- Introduce figures and tables in the main text and explain what they show. Always reference every image, table, and similar element using \ref{}.
- Use high-quality and reliable sources.
